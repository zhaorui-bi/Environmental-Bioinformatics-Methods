# Resources

## Tooling Guidance

- [TOOL.md](https://github.com/PKU-EMBL/Environmental-Bioinformatics-Methods/blob/main/TOOL.md): practical checklist for high-quality bioinformatics tools.

## Recommended Data and Platforms

- NCBI SRA / ENA for public sequencing data
- GTDB and CheckM/CheckM2 for genome quality workflows
- Standard workflow managers: Snakemake / Nextflow

## Reproducibility Stack

- Environment: `conda` or `mamba`
- Pipeline: `snakemake` or `nextflow`
- Version control: `git` + tagged releases
- Archiving: Zenodo (for key releases)
