# Policies

## Collaboration

- Discussion is encouraged, but submission artifacts must be independently produced unless team work is explicitly required.
- Team project members should clearly document each member's contribution.

## Late Policy (Example)

- <= 48 hours late: capped at 90%.
- > 48 hours late: requires instructor approval.

## Academic Integrity

- Any external code, model, or dataset must be cited.
- Generated code or text must be reviewed and validated before submission.
