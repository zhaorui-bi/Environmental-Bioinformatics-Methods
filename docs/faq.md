# FAQ

## How similar is this to CS224n-style organization?

The structure follows the same operating pattern: clear schedule, assignment cadence, project milestones, and centralized documentation.

## Should students only submit notebooks?

No. Notebooks are optional. Reproducible scripts/workflows plus concise report are preferred.

## What if tools fail due to dependency conflicts?

Document the exact error, lock the environment version, and provide a minimal reproducible command in your submission.
