# Final Project

## Objective

Build an end-to-end environmental bioinformatics study with explicit biological question, reproducible computational workflow, and defensible conclusions.

## Milestones

1. Proposal (Week 4): question, data source, hypothesis, toolchain.
2. Midterm report (Week 7): preliminary results + failure analysis.
3. Final deliverables (Week 10): report, slides, and reproducible repository.

## Deliverables

- `project/README.md` project overview
- Reproducible pipeline and environment specification
- Final report (methods, results, limitations, future work)
- 8-12 minute presentation

## Evaluation

- Problem framing quality: 20%
- Methodological rigor: 30%
- Reproducibility and code quality: 30%
- Interpretation and discussion: 20%
