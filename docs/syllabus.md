# Syllabus

## Course Goals

By the end of the course, students should be able to:

1. Explain core concepts in environmental and metagenomic bioinformatics.
2. Build reproducible pipelines for assembly, binning, annotation, and ecological interpretation.
3. Critically evaluate tool quality, assumptions, and limitations.
4. Deliver a complete team project with clear scientific and engineering evidence.

## Prerequisites

- Introductory molecular biology and genomics.
- Basic Linux command-line usage.
- One scripting language (Python/R/Bash).

## Expected Outcomes

- A fully reproducible analysis record.
- At least one assignment-grade mini pipeline.
- A final team project repository with report and presentation.

## Grading (Recommended Baseline)

- Assignments: 45%
- Final project: 40%
- Participation and discussion: 15%

## Core Topics

1. Biological foundations and bioinformatics resources
2. Sequence similarity search and BLAST practice
3. Metagenomic assembly strategies
4. Binning, refinement, and genome-quality assessment
5. Taxonomic/functional profiling and transcriptomic basics
6. Ecological interpretation and amplicon analysis
